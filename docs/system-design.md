# 系統整體設計說明（System Design）

版本：v0.1（Development Draft）

---

## 1. 系統定位

本系統為一套應用於 Python 程式設計教學之
「影片 + 字幕驅動」AI 學習系統。

核心目標：
- 建立節點化學習流程
- 在不破壞評量效度的前提下提供練習自適性
- 將動機機制與能力分級分離
- 建立可追蹤的學習痕跡（learning trace）

---

## 2. 學習結構

- 共多個單元（預計 11 單元）
- 每單元包含多支影片
- 每影片包含多個節點（node）
- 每節點包含：
  - 記憶題（Remember）
  - 理解題（Understand）

學生需依序完成影片與單元。

---

## 3. 系統分層

前端：
- Vue.js

後端：
- Flask（Blueprint 架構）
- MongoDB

---

## 4. 分級與評量邊界

系統將練習與正式評量分離：

- 影片內練習可動態調整難度
- 單元等級僅依第一次完成結果決定
- 重複練習不回寫正式分級

---

## 5. 動機機制

- 籤筒點數
- 排行榜
- 頭像等級

以上機制僅影響參與度，不影響能力估計。
